ℹ️ Guida alla Gestione Database

📊 Panoramica

Questa pagina permette di gestire multipli database di domande. Ogni database è indipendente con le sue domande, categorie e backup.

➕ Creare un Database

  • Clicca "➕ Nuovo Database" per creare un database vuoto
  • Clicca "📤 Upload JSON" per importare un file JSON esistente

✅ Selezionare un Database

  • Clicca "✅ Seleziona" su un database per lavorare con quello
  • Il database attivo è evidenziato con un bordo verde

✏️ Rinominare un Database

  • Clicca "✏️ Rinomina" per cambiare il nome
  • Il nome viene sanitizzato automaticamente
  • I backup vengono spostati insieme al database

🗑️ Eliminare un Database

  • Clicca "🗑️ Elimina" per rimuovere un database
  • Verranno chieste due conferme per sicurezza
  • L'eliminazione include anche tutti i backup

🔄 Aggiornamento Dinamico

  • Clicca "🔄 Aggiorna" per scansionare nuovi database nella cartella
  • Il refresh avviene anche automaticamente ogni 10 secondi

📥 Download

  • Clicca "⬇️ Download" per scaricare il database come file JSON
