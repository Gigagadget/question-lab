ℹ️ Guida all'Editor Domande

📋 Panoramica delle funzionalità

  • Nuova domanda: Clicca "➕ Nuova" per creare una nuova domanda
  • Modifica domanda: Seleziona una domanda dalla lista per modificarla nel pannello destro
  • Salvataggio automatico: Le modifiche vengono salvate automaticamente (indicatore "● Auto" in alto)
  • Salva manuale: Clicca "💾 Salva" per forzare il salvataggio
  • Ricarica: Clicca "🔄 Ricarica" per ricaricare le domande dal server

🔍 Ricerca e Filtri

  • Barra di ricerca: Cerca per ID o testo della domanda
  • Filtri risposte: Filtra per "Senza risposta", "Con risposte", "Senza risposta corretta"
  • Mostra Duplicati: Visualizza solo le domande con duplicati

🏷️ Categorie

  • Gestisci categorie: Clicca "🏷️ Categorie" per aggiungere, rimuovere o rinominare domini e sottodomini
  • Rinomina: Usa il pulsante "✏️ Rinomina" per cambiare nome a una categoria (aggiorna automaticamente le domande)

📊 Statistiche

  • Panoramica: Visualizza il totale delle domande, duplicati e distribuzione per categoria
  • Statistiche risposte: Controlla domande con una sola risposta, senza risposte o senza risposte corrette

💾 Backup

  • Backup automatici: Ogni salvataggio crea un backup automatico
  • Ripristina: Seleziona un backup dalla lista per ripristinare il database

⌨️ Scorciatoie da tastiera

  • Frecce sinistra/destra: Naviga tra le domande

💡 Suggerimenti

  • Le domande con duplicati sono evidenziate in giallo nella lista
  • Puoi selezionare più domande per azioni batch (elimina, cambia categoria)
  • Il campo ID deve essere univoco: se lo cambi, viene verificato automaticamente
